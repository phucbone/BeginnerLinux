<h2 id=sudo>Sudo</h2><p><code>Sudo</code> is a very special command. On Linux, most users do not have root access by default. Without root access, you cann&rsquo;t make system level changes, e.g., installing packages, creating users.</p><p>In order to perform these operations under these accounts, you&rsquo;ll need to elevate privileges with sudo. The usage of sudo is extremely simple, just put <code>sudo</code> before your target command. For example, if you want to install tmux on Linux, you cannot do <code>apt install tmux</code>, instead, you do <code>sudo apt install tmux</code>.</p><p>A user can only use sudo command if allowed in the sudoers file.</p><p>By default, sudo command will ask you for password, but the <a href=/linux/setup/setup_a_new_linux_server/#set-up-sudo-without-password>setup here</a> will help you skip that step.</p><h2 id=file-commands>File Commands</h2><p>File system commands are the most basic commands on Linux. Here is a list of them.</p><h3 id=ls>ls</h3><p><code>ls</code> is a simple command, meaning list files. It just list files in the current directory. You can also list all the files with its attributes by doing <code>ls -alh</code>.</p><h3 id=cd>cd</h3><p><code>cd</code> tells your shell to change your current directory to the target one.</p><h3 id=pwd>pwd</h3><p><code>pwd</code> returns what the current directory is.</p><h3 id=mkdir>mkdir</h3><p><code>mkdir</code> means make directory. It makes a directory in the current directory. By default, it only create one layer of directory, but you can do <code>mkdir -p {target_dir}</code> to create multiple layers of directories.</p><h3 id=cp>cp</h3><p><code>cp</code> means copy file. It copies file from one location to another one. The format of the command is</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>cp <span class=o>{</span>src-file<span class=o>}</span> <span class=o>{</span>dst-file<span class=o>}</span>
</span></span></code></pre></div><p>where <code>{src-file}</code> and <code>{dst-file}</code> are the source and target file locations.</p><h3 id=mv>mv</h3><p><code>mv</code> means move file. You can use it to move file from one location to another.</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>mv <span class=o>{</span>src-file<span class=o>}</span> <span class=o>{</span>dst-file<span class=o>}</span>
</span></span></code></pre></div><p>Note that renaming file and moving file are the same thing.</p><h3 id=rm>rm</h3><p><code>rm</code> means remove. You can use it to remove a file, or remove a director (and its contents in it) with <code>rm -Rf {target-dir}</code>.</p><h3 id=rmdir>rmdir</h3><p><code>rmdir</code> means remove directory. You can use it to remove a directory.</p><h3 id=touch>touch</h3><p><code>touch</code> just creates an empty file in the current directory. The format of the command is</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>touch <span class=o>{</span>target-file-name<span class=o>}</span>
</span></span></code></pre></div><p>After the command, a file named as <code>{target-file-name}</code> will be created.</p><h3 id=find>find</h3><p><code>find</code> finds files in the file system. You can use it to find files with specific name patterns</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>find . -iname <span class=o>{</span>your-file-pattern-regex<span class=o>}</span>
</span></span></code></pre></div><p>or use it to find files instead of directories.</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>find . -type f
</span></span></code></pre></div><h3 id=grep>grep</h3><p><code>grep</code> searches content in files. For example, if you want to search all the lines including sweworld in a file, you can do:</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>grep sweworld <span class=o>{</span>target-file-name<span class=o>}</span>
</span></span></code></pre></div><p><code>grep</code> is a very powerful tool, use <code>man grep</code> or <code>grep -h</code> to check out details.</p><h3 id=head>head</h3><p><code>head</code> shows the first few lines of a file, e.g., the following command shows the first 10 lines in the target file.</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>head -n <span class=m>10</span> <span class=o>{</span>target-file-name<span class=o>}</span>
</span></span></code></pre></div><h3 id=tail>tail</h3><p><code>tail</code> shows the last few lines of a file, e.g., the following command shows the last 10 lines in the target file.</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>tail -n <span class=m>10</span> <span class=o>{</span>target-file-name<span class=o>}</span>
</span></span></code></pre></div><p>One of the important usage of tail is to print the new content of the file as the file is being appended with new content, which is done with the following command</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>tail -f <span class=o>{</span>target-file-name<span class=o>}</span>
</span></span></code></pre></div><h3 id=tar>tar</h3><p><code>tar</code> can be used to pack a directory of content into a single file.</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>tar -cvf <span class=o>{</span>target-tar-file-name<span class=o>}</span>.tar <span class=o>{</span>target-dir-name<span class=o>}</span>
</span></span></code></pre></div><p>For example, if your <code>{target-dir-name}</code> is <code>sweworld</code>, and <code>{target-tar-file-name}</code> is <code>sweworld.tar</code>, it&rsquo;ll pack all the content in the <code>sweworld</code> dir into a file called <code>sweworld.tar</code>.</p><p>To unpack the .tar file, do</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>tar -xvf <span class=o>{</span>target-tar-file-name<span class=o>}</span>.tar
</span></span></code></pre></div><p><code>tar</code> is useful whenever you want to transfer files over network, because transferring one single file is a lot faster than transfering many files.</p><h3 id=chmod>chmod</h3><p><code>chmod</code> modifies the access of a file. The Linux file system is delicated designed and needs another article to understand.</p><h3 id=chown>chown</h3><p><code>chown</code> changes the owner of a file.</p><h3 id=zip--unzip>zip & unzip</h3><p><code>zip</code> and <code>unzip</code> will compress/decompress a file.</p><p>The usage of <code>zip</code> is:</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>zip <span class=o>{</span>target-zipfile<span class=o>}</span> file1 file2 file3 ...
</span></span></code></pre></div><p>The usage of <code>unzip</code> is:</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>unzip <span class=o>{</span>target-zipfile<span class=o>}</span>
</span></span></code></pre></div><h2 id=disk-commands>Disk Commands</h2><h3 id=df>df</h3><p><code>df</code> presents the free disk space for each block device. For example, the following command shows the disk usage of all the local disks.</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>df -alh
</span></span></code></pre></div><h3 id=du>du</h3><p><code>du</code> displays the disk usage stats.</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>du -h ./*
</span></span></code></pre></div><p>For example, the above command lists the disk usage of all the directories and files in the current directory.</p><h2 id=process-commands>Process Commands</h2><h3 id=ps>ps</h3><p><code>ps</code> lists the current running process information, including PID, TTY, running time, the command used to launched the process.</p><h3 id=kill>kill</h3><p><code>kill</code> is used to kill a process. <code>kill</code> kills a process by sending a SIGTERM signal. If that won&rsquo;t kill a process, use</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl><span class=nb>kill</span> -9 <span class=o>{</span>process-id<span class=o>}</span>
</span></span></code></pre></div><p>to send a SIGKILL signal, which is the strongest process killing signal.</p><h3 id=top>top</h3><p><code>top</code> lists all process in a terminal UI. A fancy version of <code>top</code> is called <code>htop</code>, which needs installation.</p><h2 id=network-commands>Network Commands</h2><h3 id=curl>curl</h3><p><code>curl</code> can be used to download data from a server. For example, you can use</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>curl https://sweworld.net
</span></span></code></pre></div><p>to download the main page of sweworld.</p><h3 id=netstat>netstat</h3><p><code>netstat</code> shows the network status. It is typically used to check what ports are open on the target machine.</p><h3 id=ipconfig>ipconfig</h3><p><code>ipconfig</code> shows the interface configurations. It is typically used to check the IP address of a machine.</p><h3 id=ping>ping</h3><p><code>ping</code> checks if it can send ICMP packets to a target IP address. It is typically used to check the network connectivity between your server and another one.</p><h2 id=user-commands>User Commands</h2><h3 id=useradd--userdel>useradd & userdel</h3><p><code>useradd</code> and <code>userdel</code> are used to create and delete users.</p><h2 id=package-commands>Package Commands</h2><h3 id=apt>apt</h3><p><code>apt</code> is advanced package tool. It is used to install packages on your Linux. For example, you can use the following command to install tmux</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>sudo apt install tmux
</span></span></code></pre></div><p>Note that it is typically used with sudo, since it requires root access.</p><h2 id=system-commands>System Commands</h2><h3 id=uname>uname</h3><p><code>uname</code> displays information about your system, e.g.,</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>uname -a
</span></span></code></pre></div><h3 id=hostname>hostname</h3><p><code>hostname</code> shows the hostname of the machine.</p><h3 id=systemctl>systemctl</h3><p><code>systemctl</code> is used to manage services.</p><h3 id=reboot>reboot</h3><p><code>reboot</code> command reboots your machine. If you want to reboot your machine right away, type in</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>sudo reboot now
</span></span></code></pre></div><h3 id=shutdown>shutdown</h3><p><code>shutdown</code> command shutdown your machine. If you want to shutdown your machine right away, type in</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>sudo shutdown -h now
</span></span></code></pre></div><h2 id=other-commands>Other Commands</h2><h3 id=echo>echo</h3><p><code>echo</code> just prints out what it is given.</p><h3 id=man>man</h3><p><code>man</code> means manual. It presents you the manual of another tool, e.g.,</p><div class=highlight><pre tabindex=0 class=chroma><code class=language-bash data-lang=bash><span class=line><span class=cl>man grep
</span></span></code></pre></div><p>shows your how to use the command grep.</p>
